# Persistent Sage 0.2.0-beta.4 — Rebrand release (Windows)

First release under the product name **Persistent Sage** (formerly codename Nova).

## Download

| File | Use |
|------|-----|
| **`Persistent Sage_*_x64-setup.exe`** | Recommended — desktop install |
| **`PersistentSagePortable.zip`** | USB/portable — run **`Start-Persistent-Sage-Portable.bat`** |

**SmartScreen:** Unsigned beta — **More info → Run anyway** if prompted.

## Upgrading from Nova 0.2.0-beta.3

Data moves to a new folder (`Persistent Sage` under AppData). Copy your old `data` contents or see **[REBRAND.md](../REBRAND.md)**.

## What's new

- New logo, splash screen, and sidebar branding
- Product renamed throughout the app and installer
- Same features as 0.2.0-beta.3 (memory, onboarding, NSIS installer)
