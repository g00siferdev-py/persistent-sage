# Persistent Sage 0.2.0-beta.9 — Pulse, fixes, and What's new

This beta ships the Pulse and provider fixes that landed on `main` after `0.2.0-beta.8`, plus an in-app **What's new** dialog after updates.

## What's new

- **Pulse** — scheduled background check-ins on your open chat thread; optional tools when enabled in Settings.
- **Send Pulse now** — run a check-in immediately from Settings → General.
- **Pulse in chat** — replies appear as `Pulse Response : [timestamp] - …` (your Pulse instructions stay hidden).
- **What's new dialog** — shown once after you update to a new version.
- **OpenAI API key fix** — saving an OpenAI key no longer corrupts the Ollama key slot.
- **Ollama Cloud models** — separate model list from local Ollama in Settings.

## Updating

Users on `0.2.0-beta.8` (or earlier) can update from inside the app:

**Settings → General → Updates → Check for updates**

Publish this release as a **normal** GitHub release (not a GitHub prerelease) so the updater can read `latest.json`.
