# Persistent Sage 0.2.0-beta.7 — Open beta feedback flow

This beta adds the public open beta feedback path inside Persistent Sage. Users can now open structured GitHub Issues from the app without the app automatically attaching private chats, Memory Anchors, logs, or API keys.

## What's new

- Settings → General → Open beta feedback.
- Prefilled GitHub Issue links for bug reports, ideas, and general beta feedback.
- GitHub Issue templates for structured beta reports.
- Public `PRIVACY.md`, `SECURITY.md`, and signing/update documentation.
- Documentation updates for the no-cost GitHub-first open beta path.

## Update path

Users on `0.2.0-beta.6` can use **Settings → General → Updates → Check for updates** after this release is published.

## Important

Publish this release as a normal GitHub release, not a GitHub prerelease. Persistent Sage's updater uses GitHub's `/releases/latest/download/latest.json` endpoint, which excludes GitHub prereleases.
