# Persistent Sage 0.2.0-beta.6 — Updater-enabled beta

This beta is the first updater-enabled Persistent Sage build. Existing beta users must install this version manually once. After that, future updater-compatible releases can be checked from **Settings → General → Updates**.

## What's new

- Tauri updater integration with signed update packages.
- `latest.json` release metadata generated by GitHub Actions.
- In-app **Check for updates** and **Download & install** controls.
- Privacy, security, and signing/update documentation for public project readiness.

## Important updater note

Updater-enabled beta releases must be published as normal GitHub releases, not GitHub prereleases. GitHub's `/releases/latest` endpoint excludes prereleases, and Persistent Sage uses that endpoint to fetch `latest.json`.

The version number still identifies the release as beta: `0.2.0-beta.6`.

## Windows signing

Persistent Sage is still unsigned on Windows. SmartScreen may show an unknown-publisher warning until Windows code signing is available. Future signed builds may be signed through SignPath Foundation / SignPath.io if the project is accepted.

Download the normal Windows installer:

Keep these assets attached to the release for the updater:

- `Persistent.Sage_0.2.0-beta.6_x64-setup.exe.sig`
